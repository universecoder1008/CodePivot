# 🚀 CodePivot

> Your complete placement preparation platform for coding, interview practice, and career growth.

CodePivot is a full-stack web application designed to help students prepare for software engineering placements. It provides coding practice, mock assessments, AI-powered assistance, progress tracking, and the latest hiring updates—all in one place.

---

## ✨ Features

- 🔐 Secure Authentication (Login/Register)
- 💻 Coding Practice
- 📊 Progress Tracking Dashboard
- 🤖 AI Coding Assistant
- 📝 Mock Tests & Assessments
- 📚 Interview Preparation Resources
- 📰 Latest Hiring & Placement News
- 👤 User Profile Management
- 📈 Performance Analytics

---

## 🚀 Getting Started

### Clone the Repository

```bash
git clone https://github.com/yourusername/CodePivot.git

cd CodePivot
```

---

## Install Dependencies

### Root

```bash
npm install
```

### Frontend

```bash
cd front
npm install
```

### Backend

```bash
cd ../back
npm install
```

---

## Environment Variables

Create a `.env` file inside the **back** directory.

```env
PORT=5000

MONGO_URI=your_mongodb_connection_string

JWT_SECRET=your_secret_key
```

---

## Running the Project

### Run Frontend

```bash
cd front
npm run dev
```

### Run Backend

```bash
cd back
npm run dev
```

Or run both together from the root:

```bash
npm run dev
```
